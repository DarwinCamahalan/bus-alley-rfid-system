# Automated Bus Alley Management System using RFID
---

> Components:
  * ESP32 WROOM 32 (NODE MCU)
  * MFRC522 RFID READ
  * MIFARE 1KB CLASSIC RFID CARD
  * SERVO MOTOR
  * 16X2 LCD SCREEN
  * THERMAL PRINTER
  
> CONFIGURATION:

| PIN NO.      | LABEL       |
| -----------  | ----------- |
| GND          | GND         |
| 3V3          | VIN         |
| 22           | RST  (RED)  |
| 19           | MISO (GREEN)|
| 23           | MOSI (BLUE) |
| 18           | SCK  (YELLOW)|
| 5            | SDA  (BROWN)|
