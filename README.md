# Automated Bus Alley Management System using RFID


> COMPONENTS:
  * ESP32 WROOM 32 (NODE MCU)
  * MFRC522 RFID READER
  * MIFARE 1KB CLASSIC RFID CARD
  * SERVO MOTOR
  * I2C 16X2 LCD SCREEN
  * THERMAL PRINTER

---
